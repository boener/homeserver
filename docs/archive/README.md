# 📦 Archive

This directory contains older documentation snapshots from earlier phases of the home server project.

These files are preserved for historical context, learning history, and recovery of earlier assumptions.

## Important Rule

Files in this directory are **not the current source of truth**.

Use these documents for:
- build history
- previous architecture stages
- understanding how the system evolved

Do **not** rely on them as canonical current-state documentation.

## Canonical Current Docs

Use these instead:
- `docs/current-state.md`
- `docs/architecture.md`
- `docs/changelog.md`
- `docs/services/`

## Notes

Many archived documents reflect real states the server passed through, including:
- WiFi-era networking
- early HTTPS setup
- initial NAS and Samba setup
- first Flask/Caddy integration stages

That history is useful, but it is separate from current reality.
