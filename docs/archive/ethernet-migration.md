# 🔌 Ethernet Migration (Archived)

This document describes the transition from WiFi to Ethernet.

---

## Key Phase

- Migration to wired networking
- WiFi disabled at driver level
- Stable LAN IP established

---

## Historical Context

At the time:
- Ethernet used built-in 10/100 NIC
- USB gigabit adapter had not yet been introduced

---

## Superseded By

Current system now uses:
- USB 3.0 gigabit adapter as primary interface
- fallback NIC strategy

---

Archived for reference.
